## 👋 프로젝트 소개 
<br>

* 반려 동물 소유자에게 반려동물 관련 정보를 제공하는 💜 <br>
  원스톱 플랫폼입니다. 🐶🐶

  <br> 

## 🌳 개발 내용
-사용자 회원 가입/로그인 및 로그아웃/비밀번호 재설정
<br>
-위치기반 주변 반려동물 시설 정보 제공 
<br>
-알러지 맞춤 제품 추천 기능
<br>
-앨범 게시판 기능
