## 👋 프로젝트 소개 

반려 동물 소유자에게 반려동물의 질병 정보를 주는!💜 <br>
원스톱 플랫폼입니다. 🐶🐶<br>

## 🌳 개발 내용
-사용자 회원 가입/로그인 및 로그아웃/비밀번호 재설정<br>
-위치기반 주변 반려동물 시설 정보 제공 <br>
-알러지 맞춤 제품 추천 기능<br>
-앨범 게시판 기능 <br>

### ✨달성 목표✨
애견인을 위한 개인 맞춤형 정보 제공 웹 페이지 개발<br>
